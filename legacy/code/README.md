# Codigo legado

Este diretorio preserva a implementacao anterior ao `plot_core`.

## Status

- este codigo e mantido apenas como referencia historica;
- novas implementacoes devem acontecer em `plot_core/`;
- cenarios oficiais devem ser definidos em `plot_core/scenarios/`;
- entrypoints oficiais devem ficar em `scripts/recipes/`.

## Compatibilidade

Arquivos na raiz do projeto com os mesmos nomes continuam existindo apenas
como wrappers temporarios para nao quebrar imports e execucoes antigas.
